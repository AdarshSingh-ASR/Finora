# Devpost submission draft

## Finora — Your money, finally legible

Finora turns any bank, card, or UPI statement into an explainable personal-finance memory. Upload a native or scanned PDF, spreadsheet, CSV, or screenshot. Gemini 2.5 Flash reads the document, normalizes messy merchant narrations, categorizes each transaction with confidence and a plain-language reason, and surfaces the few insights actually worth attention. Groq keeps text workflows available if Vertex is temporarily unavailable.

Users can correct uncertainty, sync a formatted summary and charts to their own Google Sheet, or ask Codex questions about the ledger through Finora's MCP server and reusable skill. CSV works in a no-key demo path; multimodal statements use the Responses API with strict Structured Outputs.

### What makes it different

It is bank-agnostic, explainable, correction-friendly, and agent-native. Instead of trapping a dashboard inside another finance app, Finora creates one clean financial memory that can travel to the tools people already use.

### Built with

Codex, Gemini 2.5 Flash on Vertex AI, Groq GPT-OSS 20B fallback, structured outputs, Model Context Protocol, Next.js/vinext, Cloudflare Sites, Google Apps Script, and Google Sheets.
